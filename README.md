# ylgy
羊了个羊JS无脑无限通关版

## 一、抓包获取t(羊了个羊身份信息)(偷懒直接引用大佬教程)

抓取 `cat-match.easygame2021.com` 的请求包内 Header 为 `t` 的内容

### 1、电脑

[Charles抓包教程](https://www.jianshu.com/p/ff85b3dac157)

### 2、IOS

[Stream 保姆级教程](https://github.com/zc2638/ylgy/blob/main/docs/stream.md)

## 三、安装axios依赖 && 执行

命令行依次输入 

```shell
npm install
```
```shell
npm run start
```
